= TXNeko

A port of XNeko to TXLin.

== Building

You'll need TXLin, Cairo, pkg-config and GCC/clang for this.

[source,bash]
----
$ sh make.sh
$ ./txneko
----

== Usage
``txneko [-i<step length in pixels>] [-s<delay in ms>] [-w<width>] [-h<height>]
